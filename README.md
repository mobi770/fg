# 🛡️ Family Guardian

בקרת גלישה משפחתית עם שליטה מרחוק מנייד

---

## 📦 מה כלול

```
family-guardian/
├── extension/          ← תוסף כרום (מותקן על המחשב בבית)
│   ├── manifest.json
│   ├── background.js   ← לוגיקת חסימה
│   ├── popup.html/js   ← ממשק התוסף
│   ├── blocked.html    ← דף "אתר חסום"
│   └── icons/
├── server/             ← שרת Node.js (מותקן ב-VPS/Render/Railway)
│   ├── server.js
│   └── package.json
└── web-dashboard/      ← לוח בקרה (נגיש מנייד)
    └── index.html
```

---

## 🚀 הוראות התקנה

### שלב 1: העלאת השרת לאינטרנט

**אפשרות א – Railway (קל ביותר, חינמי):**
1. לך ל-https://railway.app ורשום חשבון
2. לחץ "New Project" → "Deploy from GitHub repo"
3. העלה את תיקיית `server/` ל-GitHub קודם
4. Railway יתן לך URL כמו `https://my-app.railway.app`

**אפשרות ב – Render (חינמי):**
1. לך ל-https://render.com
2. New Web Service → העלה את תיקיית `server/`
3. Build command: `npm install`
4. Start command: `node server.js`

**אפשרות ג – הרצה מקומית (לבדיקה):**
```bash
cd server
npm install
node server.js
# השרת רץ על http://localhost:3000
```

---

### שלב 2: התקנת תוסף הכרום

1. פתח כרום → כתובת: `chrome://extensions/`
2. הפעל "מצב מפתח" (Developer mode) בפינה הימנית עליונה
3. לחץ "טען תוסף לא ארוז" (Load unpacked)
4. בחר את תיקיית `extension/`
5. התוסף יופיע בסרגל הכרום

---

### שלב 3: חיבור התוסף לשרת

1. לחץ על אייקון התוסף 🛡️ בכרום
2. לחץ על לשונית "⚙️ הגדרות"
3. הדבק את כתובת השרת שקיבלת (למשל: `https://my-app.railway.app`)
4. לחץ "שמור כתובת שרת"
5. לחץ "סנכרן עכשיו"

---

### שלב 4: גישה מהנייד

פשוט פתח את כתובת השרת בנייד:
```
https://my-app.railway.app
```

זהו! הלוח בקרה יפתח בדפדפן הנייד ותוכל:
- להפעיל/לכבות חסימת אינטרנט כללית
- להוסיף/להסיר אתרים מהרשימה
- להגדיר שעות מותרות לכל אתר
- כל שינוי יסונכרן לתוסף תוך כ-2 דקות

---

## ⚙️ איך זה עובד

```
[נייד/מחשב שלך] → [לוח בקרה באינטרנט] → [שרת]
                                                    ↕ (כל 2 דקות)
                         [תוסף כרום על מחשב הילדים]
```

1. **תוסף הכרום** בודק את השרת כל 2 דקות לשינויים
2. **שינוי מהנייד** נשמר בשרת מיד
3. **התוסף מסנכרן** ומפעיל/מבטל חסימות בהתאם
4. **חסימת "לפי שעות"** = האתר חסום בכל הזמן **חוץ** מהשעות שהגדרת

---

## 📱 שימוש בנטיפליי (Netlify)

אם רוצה לאחסן ב-Netlify:

1. הצב את `web-dashboard/index.html` ב-Netlify
2. השרת Node.js חייב להיות **בנפרד** (Railway/Render/VPS)
3. ערוך בקובץ `web-dashboard/index.html` את שורת:
   ```js
   const SERVER = window.location.origin;
   ```
   ושנה ל:
   ```js
   const SERVER = 'https://your-api-server.railway.app';
   ```

## 🔒 הגדרות חסימה

| סוג | התנהגות |
|-----|---------|
| **לפי שעות** | האתר חסום בכל הזמן, **פתוח רק** בשעות שהגדרת |

| **תמיד פתוח** | לא חסום |
| **חסימה כללית** | כל האינטרנט חסום מיד |

## 🛠️ פתרון בעיות

**התוסף לא חוסם?**
- ודא שהתוסף מופעל (enabled) ב-`chrome://extensions/`
- לחץ על התוסף → "סנכרן עכשיו"

**הלוח בקרה לא מתחבר?**
- ודא שהשרת רץ
- בדוק שהכתובת נכונה (https, לא http)

**השינויים לא מסונכרנים?**
- הסנכרון קורה כל 2 דקות אוטומטית
- לסנכרון מיידי: לחץ "סנכרן עכשיו" בתוסף
